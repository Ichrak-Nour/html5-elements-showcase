# 🌐 HTML5 Elements Showcase

A beginner-friendly project demonstrating the most common HTML5 elements through practical examples.

## 📖 About

This project was created while learning the fundamentals of HTML5. It showcases many of the most commonly used HTML elements, including multimedia, forms, tables, text formatting, hyperlinks, and more.

## ✨ Features

- HTML5 document structure
- Metadata
- Headings and paragraphs
- Hyperlinks
- Images
- Audio
- Video
- Text formatting
- Lists
- Tables
- Buttons
- Forms
- Iframes

## 🛠️ Technologies Used

- HTML5

This project helped me practice:

- Creating forms
- Working with tables and lists
- Understanding HTML5 fundamentals
